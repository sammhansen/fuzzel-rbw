# Installation
## Build from source
### Prequisites
- You need rust installed
  
- Clone this repo
  ```
  git clone https://github.com/sammhansen/fuzzel-rbw.git
  ```
- Build
  ```
  cargo build --release
  ```
- Copy the binary to `/usr/bin/`
  ```
  sudo cp target/release/frbw /usr/bin/
  ```
